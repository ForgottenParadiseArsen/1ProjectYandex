## Encyclopedia about tanks

### Проект создан на PyQt6/создатель - Арсен Иванов
### Цель проекта: изучение характеристик и внешнего вида танков СССР в период с 1924-1940

## Задачи проекта "Encyclopedia about tanks":
# 1) Разработать удобный интерфейс
# 2) Сделать полный режим приложению
# 3) Сделать адаптирующие картинки
# 4) Расширить список техники


## Приложение "Encyclopedia about tanks" включает в себя:
# 1) Технические характеристики танков
# 2) Курсор
# 3) Картинки танков
# 4) Даты создания этих танков
# 5) Краткое описания танков
# 6) CSS
# 7) Виджеты
